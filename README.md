# Refresh
## 功能描述（JDK1.8）
Refresh是一个支持下拉刷新和上拉加载的智能库，支持嵌套ListView、GridView、RecyclerView，还可以自定义刷新布局和加载布局，
只需实现Header和Footer接口即可。Refresh还支持实现回弹效果，支持WebView、ScrollView。
 
## 版本描述
